## 1. 営業所別売上

### 概要

ドライバーごとの月間売上・費用を一覧表示するレポートです。

### アクセス方法

| 方法           | 詳細                                                   |
| -------------- | ------------------------------------------------------ |
| URL            | `/tbm/eigyosho`                                        |
| ナビゲーション | 画面上部メニュー → **「レポート②」** → **「営業所別売上」** |
| 対象ユーザー   | 事務員、所長、システム管理者                           |

### 表示項目

| 項目             | 説明                           | 色     |
| ---------------- | ------------------------------ | ------ |
| CD               | ドライバーコード               |        |
| ドライバ         | ドライバー名                   |        |
| 件数             | 運行件数                       |        |
| 高速代（郵便）   | 郵便向け高速道路利用料         | 赤系   |
| 高速代（一般）   | 一般向け高速道路利用料         | 青系   |
| 合計（高速代）   | 高速代合計                     |        |
| 通行料（郵便）   | 郵便向け通行料請求額           | 赤系   |
| 通行料（一般）   | 一般向け通行料                 | 青系   |
| 合計（通行料）   | 通行料合計                     |        |
| 高速台30％       | 高速代の30%                    |        |
| 高速代-通行料    | 乗務員負担高速代               |        |
| 高速超過額       | 高速代超過分                   |        |
| 給与算定運賃高   | 給与計算用運賃                 |        |
| 当月運賃（円）   | 運賃 + 付帯作業料金            |        |
| 当月売上高       | 通行料 + 運賃                  |        |
| 当月燃料使用量   | 給油量合計（L）                |        |
| 当月燃料代       | 燃料費合計（円）               |        |
| 洗車機代         | 洗車機利用料合計               |        |
| 当月走行距離     | 走行距離合計（km）             |        |
| 有料合計         | 有料道路利用料合計             |        |

### データソース

- `fetchUnkoMeisaiData`: 運行明細データ
- `fetchRuisekiKyoriKichoData`: 累積距離データ
- `tbmCarWashHistory`: 洗車機利用履歴

## 5. 給与算定

### 概要

ドライバーごとの給与計算に必要な各種項目を集計するレポートです。

### アクセス方法

| 方法           | 詳細                                                 |
| -------------- | ---------------------------------------------------- |
| URL            | `/tbm/kyuyoSantei`                                   |
| ナビゲーション | 画面上部メニュー → **「レポート②」** → **「給与算定」** |
| 対象ユーザー   | **所長、システム管理者のみ**                         |

> ⚠️ 事務員はアクセス不可

### 表示項目

| 項目                   | 説明                                           |
| ---------------------- | ---------------------------------------------- |
| 出勤日数               | 勤務状況から集計した出勤日数                   |
| 有給休暇               | 有給休暇取得日数                               |
| 乗務員名               | ドライバー名                                   |
| コード                 | ドライバーコード                               |
| 車番/車種              | 使用車両のナンバー・車種                       |
| 当月運賃高（円）       | 給与算定運賃（R_JomuinUnchin）                 |
| 当月給油量（L）        | 総給油量                                       |
| 当月ss単価             | 軽油単価（営業所月次設定から）                 |
| 当月燃料代（円）       | 燃料費合計                                     |
| 洗車機料               | 洗車機利用料合計                               |
| （売上-燃料他）*0.5    | 売上から燃料代を引いた額の50%                  |
| 高速代（円）           | 乗務員負担高速代                               |
| 有給補填               | 出勤日数 × 8,800円                             |
| その他①②             | KyuyoTableRecordからの追加項目                 |
| 総支給額（円）         | 計算結果の総支給額                             |
| 食費（円）             | KyuyoTableRecordからの食費                     |
| 前払金（円）           | KyuyoTableRecordからの前払金                   |
| 高速超過負担（円）     | 高速代超過額                                   |
| 宿泊費（円）           | (出勤日数+有給日数) × 2,000円                  |
| 明細給与（円）         | 計算結果の明細給与                             |
| 売上損失利益           | 明細給与 - 車両費用                            |
| 車両・整備代           | TbmVehicle.maintenanceの合計                   |
| 保険代                 | TbmVehicle.insuranceの合計                     |
| 油脂・タイヤ・備品代   | TbmVehicle.oilTirePartsの合計                  |
| 営業損失利益           | 売上損失利益                                   |
| 営業損失利益率         | 営業損失利益 / 当月運賃高                      |
| 備考                   | 計算レート（デフォルト0.5）                    |

### 計算ロジック

```typescript
// 主要計算式
const H_tougetsuUnchinZandaka = MEIAI_SUM(`R_JomuinUnchin`)  // 当月運賃高
const K_tougutsuNenryoDai = RUISEKI_SUM(`fuelCostInPeriod`)  // 当月燃料代
const AD_rate = myKyuyoTableRecord?.rate ?? 0.5             // レート（デフォルト50%）
const M_half = (H_tougetsuUnchinZandaka - K_tougutsuNenryoDai) * AD_rate

const R_total = M_half - N_kosukudai + O_yukyuHoten + P_other1 + Q_other2  // 総支給額
const W_misyu = H_tougetsuUnchinZandaka - K_tougutsuNenryoDai - M_half - P_other1 - Q_other2  // 明細給与
```

### 関連テーブル

| テーブル名        | 用途                           |
| ----------------- | ------------------------------ |
| KyuyoTableRecord  | ドライバー別の給与設定項目     |
| UserWorkStatus    | 勤務状況（出勤/有給等）        |
| TbmVehicle        | 車両費用（整備代、保険代等）   |

## 共通仕様

### 月切り替え

すべてのレポートで画面上部に `NewDateSwitcher` コンポーネントがあり、表示月を変更できます。

### データ取得

- 承認済み（`approved: true`）の配車データのみが集計対象
- `TbmReportCl.allowNonApprovedSchedule` フラグで未承認データも含めることが可能

### 表示形式

- `CsvTable` コンポーネントで表形式表示
- 横スクロール対応（max-width: 95vw）
- 縦スクロール対応（max-height: 80vh）
