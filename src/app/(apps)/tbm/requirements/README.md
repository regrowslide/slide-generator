# TBM（運行管理システム）ドキュメント一覧

## 概要

TBM（運行管理システム）は、運送業務の配車管理、運行記録、請求書発行、給与算定などを行うシステムです。

---

## ドキュメント構成

各機能ごとに以下の構成でドキュメントを作成しています：

1. **ユーザー目線の機能一覧** - お客様向けの利用マニュアル
2. **設計者目線の詳細仕様** - 技術者向けの実装詳細
3. **テーブル定義書** - 関連するDBテーブルの定義

```
運行管理（TBM）

│   ├── 運行入力
│   └── 月間予定
├── 営業所メニュー（事務員・所長向け）
│   ├── 営業所設定
│   ├── 配車設定
│   └── ETC連携
├── レポート①
│   ├── 運行明細
│   ├── 燃費管理
│   ├── 累積距離記帳
│   └── 請求書発行
├── レポート②
│   ├── 営業所別売上
│   ├── 荷主別売上
│   ├── 車両別売上
│   ├── 運行回数
│   ├── 給与算定（所長・管理者のみ）
│   ├── 出退勤管理
│   └── 簡易走行記録（PDF）
└── 共通設定（システム管理者のみ）
    ├── 営業所
    ├── ユーザー
    ├── 車両
    ├── 荷主
    ├── カレンダー
    └── 権限管理
```

## 権限レベル

| 権限                            | アクセス可能範囲             |
| ------------------------------- | ---------------------------- |
| ドライバー（login）             | ドライバーメニューのみ       |
| 事務員（isJimuin）              | 営業所メニュー、レポート全般 |
| 所長（isShocho）                | 事務員の権限 + 給与算定      |
| システム管理者（isSystemAdmin） | 全機能 + 共通設定            |
