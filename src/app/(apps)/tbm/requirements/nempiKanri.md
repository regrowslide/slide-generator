## 1. ユーザー目線の機能一覧

### 概要

燃費管理ページでは、車両ごとの月間燃費データを確認できます。給油履歴から走行距離と給油量を計算し、燃費と燃料費を算出します。

### アクセス方法

| 方法           | 詳細                                                 |
| -------------- | ---------------------------------------------------- |
| URL            | `/tbm/nempiKanri`                                    |
| ナビゲーション | 画面上部メニュー → **「レポート①」** → **「燃費管理」** |
| 対象ユーザー   | 事務員（isJimuin）、所長（isShocho）、システム管理者 |

> 💡 ナビゲーションには「💰 燃費管理」アイコン付きで表示されます

---

### 機能一覧

#### 1.1 月切り替え

画面上部の日付切り替えボタンで表示月を変更できます。

| 操作       | 説明                 |
| ---------- | -------------------- |
| 「◀」ボタン | 前月のデータを表示   |
| 「▶」ボタン | 翌月のデータを表示   |
| 月選択     | カレンダーから月選択 |

#### 1.2 車両別燃費サマリー

各車両のカード上部に月間サマリーを表示します。

| 項目       | 説明                                     |
| ---------- | ---------------------------------------- |
| 総走行距離 | 当月の最初の給油から最後の給油までの走行距離 |
| 総給油量   | 当月の合計給油量（リットル）             |
| 燃費       | 総走行距離 ÷ 総給油量（km/L）            |
| 金額       | 総給油量 × 軽油単価                      |

> ⚠️ 給油履歴がない車両はサマリー表示が薄くなります

#### 1.3 給油履歴一覧

車両ごとに当月の給油履歴を一覧表示します。

| 列名         | 説明                                     |
| ------------ | ---------------------------------------- |
| 日付         | 給油日                                   |
| 給油時走行距離 | 給油時点のオドメーター値               |
| 区間距離     | 前回給油時からの走行距離                 |
| 給油量       | 給油量（リットル）                       |
| 燃費         | 区間距離 ÷ 給油量（km/L）                |

---

### 機能間の関連性

| 関連機能       | 関連内容                                         |
| -------------- | ------------------------------------------------ |
| 営業所設定     | 「ガソリン・軽油【月別】」タブで軽油単価を設定   |
| 車両詳細       | 給油履歴の登録・編集は車両詳細画面で実施         |
| 累積距離記帳   | オドメーター入力データとの連携                   |
| 給与算定       | 燃料費を給与計算に使用                           |

## 2. 設計者目線の詳細仕様

### 2.1 コンポーネント構成

```
nempiKanri/
└── page.tsx    # サーバーコンポーネント（ページエントリ + 表示）
```

### 2.2 データフロー

```
page.tsx (RSC)
    ↓ [dateSwitcherTemplate で日付パラメータ取得]
    ↓ [getTbmBase_MonthConfig で軽油単価取得]
    ↓ [getNenpiDataByCar で車両別燃費データ取得]
    ↓ [vehicleList から給油履歴取得]
    ↓ [AutoGridContainer で車両カード表示]
    ↓ [CsvTable で給油履歴一覧表示]
```

### 2.3 URLクエリパラメータ

| パラメータ | 型   | デフォルト | 説明       |
| ---------- | ---- | ---------- | ---------- |
| from       | Date | 月初       | 表示開始日 |
| to         | Date | 月末       | 表示終了日 |

### 2.4 燃費計算ロジック

```typescript
// 区間距離の計算
const kukanKyori = current.odometer - prev.odometer
// ※区間距離が負または0の場合は null を返す

// 燃費の計算（区間ごと）
const nempi = kukanKyori / kyuyuryo

// 月間総走行距離
const sokoKyoriInPeriod = lastOdometer - firstOdometer

// 月間平均燃費
const heikinNempiInPeriod = sokoKyoriInPeriod / sokyuyuRyoInPeriodForAverageCalculation
// ※区間距離が計算できる給油分のみを平均燃費計算の分母に使用

// 月間燃料費
const fuelCostInPeriod = sokyuyuRyoInPeriod * keiyuPerLiter
// ※keiyuPerLiter は TbmBase_MonthConfig から取得（デフォルト: 160円/L）
```

### 2.5 前月データの繰り越し

当月最初の給油の「区間距離」を計算するために、前月最後の給油履歴を取得します。

```typescript
const previousFuelHistoryByCar = await prisma.tbmVehicle.findMany({
  where: { tbmBaseId },
  include: {
    TbmRefuelHistory: {
      where: { date: { lte: firstDayOfMonth } },
      orderBy: [{ date: 'desc' }],
      take: 1,  // 直近1件のみ取得
    },
  },
})
```

### 2.6 表示レイアウト

- `AutoGridContainer` を使用してレスポンシブ表示
- 画面幅に応じて1〜3列で表示
- 各カード幅: 450px

## 3. テーブル定義書

### 3.1 TbmRefuelHistory（給油履歴）

| カラム名     | 型       | 必須 | 説明                   |
| ------------ | -------- | ---- | ---------------------- |
| id           | Int      | ○    | 主キー（自動採番）     |
| tbmVehicleId | Int      | ○    | 車両ID（FK）           |
| date         | DateTime | ○    | 給油日                 |
| odometer     | Float    | ○    | 給油時オドメーター値   |
| amount       | Float    | ○    | 給油量（リットル）     |
| createdAt    | DateTime | ○    | 作成日時               |
| updatedAt    | DateTime | ○    | 更新日時               |

#### リレーション

- `TbmVehicle` → 1:N

### 3.2 TbmBase_MonthConfig（営業所月次設定）

| カラム名       | 型       | 必須 | 説明               |
| -------------- | -------- | ---- | ------------------ |
| id             | Int      | ○    | 主キー（自動採番） |
| tbmBaseId      | Int      | ○    | 営業所ID（FK）     |
| yearMonth      | DateTime | ○    | 対象年月           |
| keiyuPerLiter  | Float    | -    | 軽油単価（円/L）   |
| gasolinPerLiter | Float   | -    | ガソリン単価（円/L）|
| createdAt      | DateTime | ○    | 作成日時           |
| updatedAt      | DateTime | ○    | 更新日時           |

#### ユニーク制約

- `unique_tbmBaseId_yearMonth`: 同一営業所・同一年月に1レコード

### 3.3 TbmVehicle（車両）※燃費管理関連フィールド

| カラム名      | 型     | 必須 | 説明               |
| ------------- | ------ | ---- | ------------------ |
| id            | Int    | ○    | 主キー             |
| tbmBaseId     | Int    | ○    | 営業所ID（FK）     |
| vehicleNumber | String | -    | 車両ナンバー       |
| frameNo       | String | -    | 車台番号           |
| code          | String | -    | 車両コード         |

## 4. チェックリスト

### 導入時の確認事項

- [ ] 営業所設定の「ガソリン・軽油【月別】」で軽油単価を設定
- [ ] 車両詳細画面で給油履歴を登録

### 運用時の注意事項

- [ ] 給油時には必ずオドメーター値を記録
- [ ] オドメーター値は昇順になるよう正確に入力
- [ ] 月初の燃費計算には前月最後の給油データが必要

### 計算に関する注意

| 状況                     | 影響                                   |
| ------------------------ | -------------------------------------- |
| 給油履歴がない           | サマリー表示が薄くなり、計算値は0      |
| 軽油単価が未設定         | デフォルト値（160円/L）で計算          |
| オドメーター値が前回より小さい | 区間距離がnullとなり燃費計算から除外 |
| 前月データがない         | 当月最初の給油の区間距離が計算不可     |
