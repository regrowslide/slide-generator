# Regrow ユーザーマニュアル

## 1. アプリ概要

### 対象業態

リラクゼーションスパ「asian relaxation villa」

### 対象店舗

| 店舗名                           | 略称       |
| -------------------------------- | ---------- |
| asian relaxation villa新潟西店   | 新潟西店   |
| asian relaxation villa三条店     | 三条店     |
| asian relaxation villa新潟中央店 | 新潟中央店 |

### 目的

月次業績レポート（16枚のスライド資料）を自動生成するツールです。
Excelの「担当者別分析表」と手動入力データを組み合わせて、店舗KPI・スタッフ別パフォーマンス・先月比分析などを可視化します。

---

## 2. 画面構成と操作フロー

画面上部の **セクションタブ** で5つのセクションを切り替えます。

| セクション | タブ名     | 役割                                |
| ---------- | ---------- | ----------------------------------- |
| ガイダンス | ガイダンス | 全体の進捗確認・各ステップへの案内  |
| Excel取込  | Excel取込  | 担当者別分析表のアップロード        |
| データ確認 | データ確認 | インポート済みデータの一覧表示      |
| 手動入力   | 手動入力   | 店舗KPI・スタッフ稼働率等の手動入力 |
| スライド   | スライド   | 生成された16枚のスライド資料を閲覧  |

### 基本フロー

```
ガイダンス → Step 1: Excel取込 → Step 2: 手動入力 → Step 3: スライド閲覧
```

---

## 3. Step 1: Excelインポート

### 対象ファイル形式

```
担当者別分析表_asian relaxation villa{店舗名}_{日付}.xlsx
```

例:

- `担当者別分析表_asian relaxation villa新潟西店_20260209.xlsx`
- `担当者別分析表_asian relaxation villa三条店_20260209.xlsx`
- `担当者別分析表_asian relaxation villa新潟中央店_20260209.xlsx`

### 操作手順

1. 「Excel取込」タブを開く
2. ドラッグ＆ドロップエリアにExcelファイルをドロップ（またはクリックしてファイル選択）
3. 自動的にExcel内容が解析され、スタッフデータと店舗合計が取り込まれる
4. **3店舗分** のファイルをそれぞれアップロードする

### 注意事項

- 対応形式は `.xlsx` / `.xls` のみ
- 同じ店舗のファイルを再アップロードすると上書きされる
- アップロード後、「取込済みデータ」セクションで店舗名・スタッフ数・売上合計が確認できる

### インポート確認画面

「データ確認」タブで、インポート済みの全データを一覧表示できます。

- 店舗別のスタッフ一覧（ランキング順）
- 各スタッフの売上・客数・指名数・客単価
- 店舗合計値

---

## 4. Step 2: 手動入力

「手動入力」タブには3つのサブタブがあります。

### タブ1: 店舗KPI

各店舗（3店舗）に対して以下を入力します。

| 項目       | 入力方法     | 説明                                    |
| ---------- | ------------ | --------------------------------------- |
| 稼働率 (%) | 手動入力     | 店舗の月間稼働率                        |
| 再来率 (%) | **自動計算** | Excelデータから自動算出（読み取り専用） |
| 失客率 (%) | 手動入力     | 店舗の月間失客率                        |
| CS登録数   | **自動計算** | スタッフCS登録数の合計（読み取り専用）  |
| コメント   | 手動入力     | 月次の店舗コメント                      |

### タブ2: スタッフ稼働率・CS登録数

Excelインポート後、スタッフ一覧が自動表示されます。各スタッフに対して以下を入力します。

| 項目       | 入力方法     | 説明                                       |
| ---------- | ------------ | ------------------------------------------ |
| 稼働率 (%) | 手動入力     | スタッフ個人の月間稼働率                   |
| 再来率 (%) | **自動計算** | Excelの対応客数・新規客数から自動算出      |
| CS登録数   | 手動入力     | スタッフのCS（カスタマーサービス）登録件数 |

### タブ3: お客様の声

- テキストエリアにお客様からのフィードバックを自由入力
- スライド16「お客様の声」にそのまま反映される

### 自動計算項目の説明

| 項目             | 計算式                                            |
| ---------------- | ------------------------------------------------- |
| スタッフ再来率   | (対応客数 - 新規客数) / 対応客数 x 100            |
| 店舗再来率       | 店舗全スタッフの再来客数合計 / 対応客数合計 x 100 |
| 店舗CS登録数合計 | 店舗全スタッフのCS登録数の合計                    |

### 自動保存の仕組み

- 入力欄からフォーカスが外れた時点（onBlur）で自動保存される
- 保存時に画面右上に「保存完了」が一時表示される
- データはブラウザのlocalStorageに保存される（サーバーには送信されない）

## 5. Step 3: スライド閲覧

### 16枚のスライド構成

| No. | タイトル                      | 内容                                                           |
| --- | ----------------------------- | -------------------------------------------------------------- |
| 1   | タイトルスライド              | 月次業績レポート + 対象月 + villa名                            |
| 2   | 目次                          | レポートの構成一覧                                             |
| 3   | 全体サマリー                  | 店舗別KPIテーブル + 店舗コメント                               |
| 4   | 客単価 - 年間推移             | 3店舗の客単価を月別折れ線グラフで表示                          |
| 5   | 稼働率 - 年間推移             | 3店舗の稼働率を月別折れ線グラフで表示                          |
| 6   | 再来率 - 年間推移             | 3店舗の再来率を月別折れ線グラフで表示                          |
| 7   | 失客率 - 年間推移             | 3店舗の失客率を月別折れ線グラフで表示                          |
| 8   | 全指標統合 - 年間推移         | 4指標を1つのグラフに統合して表示                               |
| 9   | スタッフ別パフォーマンス      | スタッフ別テーブル（売上・稼働率・指名割合・再来率・CS登録率） |
| 10  | スタッフ稼働率                | スタッフ稼働率の縦棒グラフ（月選択可）                         |
| 11  | スタッフ別先月比（1）テーブル | 売上金額・指名件数の今月/先月/差分テーブル                     |
| 12  | スタッフ別先月比（1）グラフ   | 売上（棒グラフ）+ 指名（折れ線）の複合グラフ                   |
| 13  | スタッフ別先月比（2）テーブル | 再来率・客単価の今月/先月/差分テーブル                         |
| 14  | スタッフ別先月比（2）グラフ   | 客単価（棒グラフ）+ 再来率（折れ線）の複合グラフ               |
| 15  | 予備ページ                    | 予備用（空白）                                                 |
| 16  | お客様の声                    | 手動入力されたお客様の声を表示                                 |

### 店舗フィルタ

- 画面下部に固定表示される **店舗フィルタ** で表示店舗を切り替えられる
- チェックボックスで新潟西店・三条店・新潟中央店を個別にON/OFFできる
- フィルタはスライド3～14に適用される

### 月選択機能（スライド10）

- スライド10「スタッフ稼働率」にはスライド内で月を切り替えられるセレクトボックスがある
- 他の月のスタッフ稼働率グラフを比較できる

### 印刷/PDF出力

- ブラウザの印刷機能（Ctrl+P / Cmd+P）で印刷またはPDF出力できる

---

## 6. 月の管理

### 月選択

- 画面上部の **MonthSelector** で対象月を切り替える
- ドロップダウンから既存の月を選択
- 2026年1月～12月が初期表示される

### 新規月作成

- MonthSelectorの「新規作成」ボタンから YYYY-MM 形式で新しい月を追加できる
- 作成された月は空のデータで初期化される

---

## 7. モックデータの使い方

ガイダンスページ下部の「開発用モックデータ」セクションから、デモ用サンプルデータを読み込めます。

### 読み込まれるデータ

- 2026年1月～12月の通年データ（12ヶ月分）
- 3店舗 x 5名 = 15名のスタッフデータ
- 季節変動を反映した売上データ
- 店舗KPI（稼働率・失客率・月別コメント）
- スタッフ稼働率とCS登録数
- 月別のお客様の声

### 注意事項

- 読み込み時に確認ダイアログが表示される
- **既存の2026年データは上書きされる**

---

## 8. データステータスの確認方法

ガイダンスページの「データステータス」セクションで以下を確認できます。

| 確認項目           | 表示内容                              |
| ------------------ | ------------------------------------- |
| インポート済み店舗 | 各店舗のインポート状況（完了/未完了） |
| 手動入力進捗       | 店舗KPI件数 / スタッフデータ件数      |
| 最終更新日時       | データの最終保存日時                  |

### 手動入力タブの警告表示

- 手動入力が未完了の場合、セクションタブに赤い **「!」** バッジが表示される

---

## 9. 完了判定条件

全ステップの完了判定は以下の通りです。

| ステップ                | 完了条件                                       |
| ----------------------- | ---------------------------------------------- |
| Step 1: Excelインポート | 3店舗分の storeTotals が存在する               |
| Step 2: 手動入力        | 3店舗分の storeKpi + 1件以上の staffManualData |
| Step 3: スライド閲覧    | Step 1 と Step 2 の両方が完了している          |

全ステップ完了時、ガイダンスページに「すべてのステップが完了しました」メッセージが表示されます。

---

## 10. データ保存について

- 全データはブラウザの **localStorage** に保存される
- サーバーへの送信は行われない
- ブラウザのキャッシュを削除するとデータが消失する
- 保存キー形式: `regrow_data_YYYY-MM`（例: `regrow_data_2026-02`）
