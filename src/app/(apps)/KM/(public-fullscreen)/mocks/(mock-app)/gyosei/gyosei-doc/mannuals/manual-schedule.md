## 1. 本ツールについて

「スケジュール秘書」は、**顧客との日程調整をAIが代行する**ツールです。

### 解決する課題

- TimeRex等の日程調整URLをそのまま送ると「機械的」「失礼だ」と受け取られる顧客がいる
- テキストで候補日を丁寧に打診したいが、手作業が多い（空き確認→仮押さえ→メール→返信待ち→確定登録）
- ダブルブッキングのリスクがある
- 返信がない場合の放置リスクがある

### できること

- Googleカレンダーの空き状況を自動確認し、**候補日を自動抽出**
- 候補日を**カレンダーに仮押さえ**（ダブルブッキング防止）
- 丁寧な文面の**日程打診メールを自動作成**
- 顧客からの返信を**自動検知**し、確定処理を実行
- 返信がない場合に**アラート通知**をChatWorkへ送信
- **再調整メール**の自動作成

### 全体ワークフロー

```
① 候補日抽出 → ② 仮押さえ → ③ 日程打診メール → ④ 返信検知・確定 → ⑤ 未返信アラート → ⑥ 再調整
```

### 現在の環境との連携

| サービス | 役割 |
|---------|------|
| **Googleカレンダー** | スケジュール管理（TimeRexの裏側） |
| **Gmail** | メール送受信（メールワイズでも同一メール受信可能） |
| **ChatWork** | アラート通知の受信先 |

## 2. 初期設定

### 2-1. 動作環境

| 項目 | 要件 |
|------|------|
| ブラウザ | Google Chrome / Edge / Safari（最新版推奨） |
| インターネット | 必須 |
| ログイン | **必須**（保利国際法務事務所スタッフアカウント） |

### 2-2. Google OAuth認証（初回のみ）

本ツールはGoogleカレンダーとGmailにアクセスするため、初回利用時にGoogleアカウントの認証が必要です。

#### 手順

1. ツールにアクセスし、ログインします
2. 「**Googleアカウントを連携する**」ボタンをクリックします
3. Googleのログイン画面が表示されます
4. 保利国際法務事務所で使用しているGoogleアカウントを選択します
5. アクセス権限の確認画面が表示されます

#### 要求される権限

| 権限 | 用途 |
|------|------|
| **Googleカレンダーの読み取り・書き込み** | 空き状況の確認、仮押さえ・確定登録 |
| **Gmailの読み取り・送信** | 日程打診メールの送信、返信の検知 |

6. 「**許可**」をクリックして認証を完了します
7. 認証成功後、メイン画面（チャットUI）が表示されます

#### 注意事項

- 認証トークンはサーバー側に暗号化して保存されます
- トークンは自動的にリフレッシュされるため、通常は再認証不要です
- セキュリティ上の理由でGoogleが再認証を求める場合があります

### 2-3. ChatWork通知の設定

未返信アラートをChatWorkで受信するための設定です。

1. 管理者にChatWork通知先のルームIDを伝えます
2. 管理者側で設定が完了すると、アラートがChatWorkに届くようになります

## 3. メイン画面（チャットUI）

ログイン後のメイン画面はチャット形式のインターフェースです。AIに自然言語で指示を出して日程調整を行います。

```

│                                           │
```

- 画面右上の「**調整一覧**」から進行中の日程調整を一覧確認できます

## 4. 候補日の抽出

### 操作手順

1. チャットに以下のように入力します：

   ```
   田中様との打ち合わせ候補日を教えて
   ```

   または

   ```
   来週の空いている日時を教えて
   ```

2. AIがGoogleカレンダーの空き状況を自動確認します
3. 候補日が3〜5件程度テキストで提示されます

### AIの応答例

```
🤖 カレンダーを確認しました。以下の候補日はいかがでしょうか？

  ① 3月15日（月）10:00〜11:00
  ② 3月16日（火）14:00〜15:00
  ③ 3月17日（水）13:00〜14:00
  ④ 3月18日（木）10:00〜11:00
  ⑤ 3月19日（金）15:00〜16:00

候補日を変更したい場合はお知らせください。
```

### 候補日のカスタマイズ

以下のような指示で候補日を絞り込むことができます：

| 指示例 | 動作 |
|--------|------|
| 「午前中だけで候補を出して」 | 午前の空き時間のみ提示 |
| 「来週の月〜水で」 | 指定曜日に限定 |
| 「1時間半の枠で」 | 指定時間幅の空きを検索 |
| 「③は除いて」 | 特定候補を除外 |

## 5. カレンダーの仮押さえ

候補日が決まったら、Googleカレンダーに仮予定を登録します。

### 操作手順

1. チャットに以下のように入力します：

   ```
   その候補日で仮押さえして
   ```

2. AIが候補日すべてをGoogleカレンダーに仮予定として登録します
3. 登録結果が表示されます

### AIの応答例

```
🤖 以下の3件を仮押さえしました。

  ✅ 3月15日（月）10:00〜11:00  → カレンダーに登録済み
  ✅ 3月16日（火）14:00〜15:00  → カレンダーに登録済み
  ✅ 3月17日（水）13:00〜14:00  → カレンダーに登録済み

メールを作成しますか？
[メール作成]  [キャンセル]
```

### 仮押さえの仕様

| 項目 | 内容 |
|------|------|
| カレンダー上の表示 | 「仮押さえ」マーク付きの予定として表示 |
| ステータス | `tentative`（仮予定） |
| 自動解除 | 日程確定時に他の仮押さえは自動解除 |
| 手動解除 | 「仮押さえを解除して」と指示することで一括解除可能 |

### 注意事項

- 仮押さえした時間帯は他の予定と重複しないよう保護されます
- 仮押さえ中の時間帯に別の予定が入った場合、AIが警告します

## 6. 日程打診メールの送信

### 操作手順

   ```
   田中様にメールを送って
   ```

2. AIがメール文面を自動作成し、**プレビュー画面**に表示します

```
│                                           │

│                                           │
```

### 各ボタンの機能

| ボタン | 機能 |
|--------|------|
| **編集** | メール文面を編集画面で修正 |
| **送信する** | Gmail経由でメールを送信 |
| **キャンセル** | メール作成をキャンセル（仮押さえは維持） |

### 重要: 人間承認の仕組み

- **メールは即時送信されません**。必ずプレビュー画面で内容を確認してから「送信する」を押してください
- これは誤送信防止のための安全機構です
- すべてのメール送信は実行記録として保存されます

### メール文面のカスタマイズ

チャットで以下のような指示を出すと、文面を調整できます：

| 指示例 | 動作 |
|--------|------|
| 「もう少し丁寧な文面で」 | 敬語レベルを上げた文面に調整 |
| 「Web会議であることも書いて」 | 会議形態を追記 |
| 「前回の御礼も入れて」 | 文面に御礼文を追加 |

## 7. 返信検知と日程確定

メール送信後、顧客からの返信をAIが自動的に検知します。

### 自動検知の仕組み

| 項目 | 内容 |
|------|------|
| 検知方式 | Gmailのポーリング（定期チェック） |
| チェック間隔 | **15分ごと** |
| 検知対象 | 送信したメールへの返信 |

### 日程確定の流れ

1. 顧客が「②の3月16日でお願いします」等と返信
2. AIが返信を検知し、日時を自然言語解析で抽出
3. 以下の処理が自動実行されます：

| 処理 | 内容 |
|------|------|
| カレンダー確定 | 3月16日の仮押さえを正式な予定（`confirmed`）に変更 |
| 他の仮押さえ解除 | 3月15日・3月17日の仮押さえを自動削除 |
| 確定通知 | ユーザー（スタッフ）にチャットまたはChatWorkで通知 |

### 確定通知の例

```
🤖 田中様との日程が確定しました。

  📅 3月16日（火）14:00〜15:00

  カレンダーを更新し、他の仮押さえ（2件）を解除しました。
```

## 8. 未返信アラート

メール送信後、翌営業日までに返信がない場合、自動でアラートが通知されます。

### 通知の仕組み

| 項目 | 内容 |
|------|------|
| 判定タイミング | 翌営業日のチェック時 |
| 営業日判定 | 土日・祝日を除外 |
| 通知先 | **ChatWork** |
| 通知内容 | 「○○様からお返事がありません」 |

### ChatWork通知例

```
【スケジュール秘書】未返信アラート

田中様への日程打診メール（3/14送信）に対して、
まだお返事がありません。

対応候補：
・リマインドメールを送信
・仮押さえを解除して再調整
・電話で直接確認

▶ アプリで確認: [リンク]
```

### アラート受信後の対応

アラートを受信したら、アプリのチャットUIで以下のいずれかを指示できます：

| 指示例 | 動作 |
|--------|------|
| 「田中様にリマインドメールを送って」 | フォローアップメールを作成 |
| 「仮押さえを解除して」 | 仮予定をすべて削除 |
| 「再調整メールを送って」 | 新しい候補日での再調整メールを作成 |

## 9. 再調整メールの送信

顧客の都合が合わない場合や、仮押さえを解除して再調整する場合の操作です。

### 操作手順

#### 仮押さえの解除

1. チャットに以下のように入力します：

   ```
   田中様の仮押さえをすべて解除して
   ```

2. AIがカレンダーの仮押さえを一括削除します

#### 再調整メールの送信

1. チャットに以下のように入力します：

   ```
   田中様に再調整メールを送って
   ```

2. AIが再調整用のメール文面を作成します
3. プレビュー画面で確認後、「**送信する**」で送信

### 再調整メールの例

```
田中様

お世話になっております。保利でございます。

先日ご連絡しました日程の件ですが、
ご都合の良い日時をお教えいただけますと幸いです。

改めて候補日をご提案することも可能ですので、
お気軽にご返信ください。

何卒よろしくお願いいたします。
```

## 10. 日程調整管理画面

進行中・完了済みの日程調整セッションを一覧で管理できます。

### アクセス方法

- メイン画面右上の「**調整一覧**」ボタンをクリックします

### 一覧画面のカラム

| カラム | 内容 |
|--------|------|
| **顧客名** | 調整相手の名前 |
| **ステータス** | 下書き / 提案中 / 確定 / キャンセル |
| **候補日数** | 仮押さえ中の候補日の件数 |
| **最終更新** | 最後にアクションがあった日時 |
| **操作** | 詳細表示 / 再調整 / キャンセル |

### ステータスの意味

| ステータス | 説明 | 色 |
|-----------|------|-----|
| **下書き** | チャットで候補日を検討中（まだメール未送信） | グレー |
| **提案中** | 日程打診メールを送信済み、返信待ち | 青 |
| **確定** | 顧客の返信により日程が確定 | 緑 |
| **キャンセル** | 調整をキャンセル（仮押さえ解除済み） | 赤 |

### 操作ボタン

| ボタン | 機能 |
|--------|------|
| **詳細表示** | チャット履歴・メール履歴・候補日一覧を確認 |
| **再調整** | この調整セッションのチャット画面に移動し、再調整を開始 |
| **キャンセル** | 調整をキャンセルし、すべての仮押さえを解除 |

## 11. よくあるご質問（FAQ）

### Q. AIが提示する候補日は正確ですか？

A. AIはGoogleカレンダーのリアルタイムの空き状況を参照しています。ただし、カレンダーに登録されていない予定（口頭での約束など）は反映されません。候補日は必ずご自身でも確認してください。

### Q. メールが自動送信されることはありますか？

A. いいえ。すべてのメール送信は**必ず人間の承認（「送信する」ボタンのクリック）を経て**から実行されます。これは初期リリースにおける安全方針です。

### Q. 顧客がメールに返信しない場合どうなりますか？

A. 翌営業日（土日祝を除く）までに返信がない場合、ChatWorkにアラート通知が届きます。その後の対応（リマインドメール、再調整、直接連絡）はご自身でご判断ください。

### Q. 仮押さえした時間帯に他の予定を入れてしまった場合は？

A. 仮押さえはGoogleカレンダー上で「仮予定」として登録されています。手動で他の予定を上書きした場合、AIが次回チェック時に競合を検出し、警告を表示します。

### Q. TimeRexとの併用は可能ですか？

A. はい。TimeRexはGoogleカレンダーを裏側で使用しているため、本ツールの仮押さえはTimeRex側でも「予定あり」として反映されます。ダブルブッキングは防止されます。

### Q. ChatWork以外で通知を受けたい場合は？

A. 初期リリースではChatWorkのみ対応です。Slack等の他サービスへの対応は将来的に検討中です。

### Q. 複数人との日程調整を同時に進められますか？

A. はい。日程調整は「セッション」単位で管理されるため、複数の顧客との調整を並行して進められます。「調整一覧」画面で進行中のすべてのセッションを確認できます。

### Q. Google認証が切れた場合は？

A. トークンは自動リフレッシュされるため、通常は再認証不要です。万一失効した場合は、ログイン画面で再度Google認証を行ってください。

## 技術仕様

| 項目 | 内容 |
|------|------|
