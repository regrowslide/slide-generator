# 顧客の声分析システム
## お客様の声を、活きた経営資源に変えるAIツール

---

## 本システムで実現すること

アンケート回答を、**AIが自動で読み取り、整理・分類**します。

### 例

> 「お店の雰囲気がとてもオシャレで居心地が良かったです。コーヒーも美味しくて、スタッフの方も親切でした。ただ、BGMがちょっと大きくて会話しづらかったのが残念でした。」

### AIが自動でこのように整理します

| お客様の言葉 | カテゴリ | 感情 |
|-------------|---------|------|
| お店の雰囲気がとてもオシャレで居心地が良かったです。 | オシャレ・雰囲気が良い | 好意的 |
| コーヒーも美味しくて、 | 料理・ドリンクが美味しい | 好意的 |
| スタッフの方も親切でした。 | 気持ちの良い接客 | 好意的 |
| BGMがちょっと大きくて会話しづらかったのが残念でした。 | BGMが大きすぎる | 不満 |

```

② 内容を理解する

━━━━━━━━━━━━━━━━━━━━━━━━━
→ 100件で約3〜4時間！
```

```
AIによる自動処理
━━━━━━━━━━━━━━━━━━━━━━━━━

② ボタンをクリック

```

## 使うと何が嬉しいの？

### 1. 圧倒的な時間短縮

| 項目 | 従来（手作業） | このシステム |
|------|--------------|-------------|
| 100件の分析 | 約3〜4時間 | 約5分 |
| 必要な人員 | 専任スタッフ | 誰でもOK |
| 分析の一貫性 | 担当者によりバラつき | 常に一定 |

### 2. 「使えば使うほど賢くなる」AI

このシステムの最大の特徴は、**AIが御社専用に育っていく**ことです。

```
┌─────────────────────────────────────┐
│   最初はちょっと間違える   →   どんどん正確に   │
│          ↓                        ↓          │
│    人が修正を教える    →    AIが学習する      │
│          ↓                        ↓          │
│    次回から正しく分類  ←   修正を覚える       │
└─────────────────────────────────────┘
```

**例：**
- 最初：「抹茶ラテが美味しい」を「料理の美味しさ」に分類
- 人が「ドリンクの美味しさ」に修正
- 次回から：「コーヒーが美味しい」も正しく「ドリンクの美味しさ」に分類

### 3. 御社専用のカテゴリで分析

既存のツールは「ポジティブ/ネガティブ」程度の大雑把な分類しかできません。

**このシステムなら：**
- 御社のビジネスに合わせた細かいカテゴリを設定可能
- 「接客・サービス」「店内環境」「料理・ドリンク」など業種に合わせた分類
- 「オシャレ・雰囲気が良い」「BGMが大きすぎる」など具体的なカテゴリも設定可能

## どうやって動いているの？

難しい技術の話は省略して、イメージでお伝えします。

### AIは「新人スタッフ」のようなもの

```
┌────────────────────────────────────────────────┐
│                                                │
│   【AIの頭の中】                                │
│                                                │
│   ① マニュアル（カテゴリ一覧）を見る              │
│      → 「どんな分類があるか」を理解              │
│                                                │
│   ② 先輩の修正メモ（過去の修正事例）を見る       │
│      → 「こういう場合はこう分類する」を学ぶ       │
│                                                │
│   ③ ルールブック（追加ルール）を見る             │
│      → 「特に気をつけるポイント」を把握          │
│                                                │
│   ④ お客様の声を読んで、分類する                │
│                                                │
└────────────────────────────────────────────────┘
```

### 「教えれば教えるほど賢くなる」仕組み

1. **AIが分析結果を出す**
2. **間違いがあれば人が修正する**
3. **修正内容が「先輩のメモ」として保存される**
4. **次回からAIが参照して、同じ間違いをしなくなる**

この繰り返しで、AIは御社の判断基準を学習していきます。

## 分析できる3つの指標

### 1. カテゴリ分類

お客様の声を「何についての意見か」で分類します。

**業種別プリセット例：**

**飲食店の場合：**
- 接客・サービス
- 店内（雰囲気・清潔感）
- 料理・ドリンク
- 備品・設備
- 値段
- 立地

**スポーツイベントの場合：**
- 試合・競技
- 座席・観覧環境
- スタッフ・サービス
- 飲食・売店
- アクセス・立地
- グッズ・物販

### 2. 感情分析

お客様の気持ちを3つ（＋1つ）に分類します。

| 感情 | 意味 | 例 |
|------|------|-----|
| 好意的 | 褒めている・喜んでいる | 「とても美味しかった！」 |
| 不満 | 不満がある・改善して欲しい | 「ちょっと遅かった」 |
| リクエスト | 要望・提案 | 「新メニューが欲しい」 |
| その他 | 上記に当てはまらない | 「2回目の来店です」 |

### 3. 熱量スコア（0〜100点）

お客様の感情の**強さ**を数値化します。

| スコア | 意味 | 例 |
|--------|------|-----|
| 0〜25 | 軽い感想 | 「まあまあでした」 |
| 26〜50 | 普通の評価 | 「美味しかったです」 |
| 51〜75 | 強い感情 | 「本当に最高！」 |
| 76〜100 | とても熱い想い | 「感動しました！！！絶対また来ます！」 |

## 使い方の流れ

### ステップ1：初期設定（最初だけ）

1. **業種を選ぶ**
   - 「飲食店」「スポーツイベント」などから選択
   - 業種に合った分類カテゴリが自動でセットされます

2. **カテゴリをカスタマイズ（任意）**
   - 必要に応じて、御社独自のカテゴリを追加・編集

### ステップ2：分析を実行

1. **テキストを入力**
   - レビューやアンケート回答をコピー＆ペースト
   - 複数件まとめて入力可能

2. **「分析実行」ボタンをクリック**
   - AIが自動で分析開始

3. **結果を確認**
   - カテゴリ・感情・熱量が表示されます

### ステップ3：修正してAIを育てる（重要！）

1. **間違いがあれば修正**
   - カテゴリや感情を正しいものに変更

2. **「フィードバック送信」ボタンをクリック**
   - 修正内容がAIの学習データとして保存されます

3. **次回以降、AIが賢くなる**
   - 同じパターンの間違いが減っていきます

## よくある質問

### Q1：AIは最初からちゃんと分類できますか？

**A：** 最初はある程度の間違いがあります。しかし、修正を重ねることで御社専用のAIに育っていきます。通常、50〜100件程度の修正を行うと、かなり正確になります。

### Q2：どのくらいの量を一度に分析できますか？

**A：** 一度に数十件〜数百件のテキストを分析できます。レビューやアンケート回答をまとめてコピー＆ペーストするだけでOKです。

### Q3：既存のカテゴリ分類を変更できますか？

**A：** はい。カテゴリはいつでも追加・編集・削除ができます。ビジネスの変化に合わせて柔軟に対応できます。

### Q4：分析結果はどこに保存されますか？

**A：** すべての分析結果と修正履歴はデータベースに保存されます。過去の分析結果を振り返ったり、データとしてエクスポートすることも可能です。

### Q5：複数の店舗やブランドで使えますか？

**A：** はい。「クライアント」という単位でデータを分離管理できるため、複数の店舗やブランドで別々に分析できます。それぞれ独自のカテゴリやルールを設定できます。
