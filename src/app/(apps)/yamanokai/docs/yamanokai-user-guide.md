# 1. はじめに

## 1.1 このシステムについて

「山の会（KCAC）システム」は、神戸勤労者山岳会の活動を一元管理するためのWebアプリケーションです。

### 従来の課題

| 課題 | 詳細 |
|------|------|
| 情報が分散 | 名簿、企画、計画書がバラバラに管理されている |
| 転記作業が多い | 計画書作成時に会員情報を何度も入力 |
| 検索できない | 過去の記録を探すのが大変 |
| 装備管理が曖昧 | 誰がいつまで借りているかわからない |

### このシステムで解決

| 解決策 | 効果 |
|--------|------|
| 情報の一元管理 | すべての情報が1つの場所に |
| 自動入力・転記 | 会員情報は選択するだけ |
| 検索・フィルタ | 過去の記録もすぐに見つかる |
| 貸出管理の見える化 | 返却期限も自動でお知らせ |

## 1.2 アクセス方法

Webブラウザ（PC・スマートフォン対応）から以下のURLにアクセスします。

```
https://[システムURL]/yamanokai
```

> **スマートフォンの場合**: ホーム画面に追加すると、アプリのように使えます。

# 3. 画面別操作ガイド

## 3.1 📅 ダッシュボード

**概要**: ログイン直後に表示される画面。重要な情報を一覧で確認できます。

### 表示される情報

| 項目 | 内容 |
|------|------|
| 会員数 | 現在の登録会員数 |
| 今月の例会 | 今月予定されている例会数 |
| 貸出中装備 | 現在貸し出されている装備数 |
| 返却期限超過 | 期限を過ぎた装備（⚠️警告表示） |
| 直近の例会案内 | 近い日程の例会案内5件 |

### こんなときに便利

- 「今週末の例会は何があるかな？」→ 直近の例会案内を確認
- 「借りた装備の返却期限は大丈夫かな？」→ 返却期限超過を確認

---

## 3.2 📋 例会企画（カレンダー）

**概要**: 年間の例会企画をカレンダー形式で確認・管理できます。

### 基本操作

| 操作 | 方法 |
|------|------|
| 月を移動 | 「← 前月」「次月 →」ボタンをクリック |
| 企画を確認 | カレンダー上の企画名をクリック |
| 企画を追加 | 「+ 企画追加」ボタンをクリック |

### 企画追加の手順

1. 「+ 企画追加」ボタンをクリック
2. 必要事項を入力
   - **日付**: 開催日を選択
   - **担当部**: ハイキング部/山行部/教育部/自然保護部/組織部
   - **タイトル**: 例会名（例: クリーンハイク）
   - **CL**: 担当リーダーを選択（未定でも可）
3. 「追加」ボタンをクリック

### 色分けの見方

| 色 | 担当部 |
|----|--------|
| 🟢 緑 | ハイキング部 |
| 🔵 青 | 山行部 |
| 🟣 紫 | 教育部 |
| 🟡 黄 | 自然保護部 |
| ⚪ グレー | 組織部 |

## 3.3 📢 例会案内

**概要**: 参加者募集中の例会案内を確認し、申し込みができます。

### 一覧画面の見方

各カードに表示される情報:

```
┌─────────────────────────────────────┐
│ [山行部]              申込: 3名     │
│ ──────────────────────────────────  │
│ 西穂高岳 2908m                      │
│ 4/12(土) 〜 4/14(月)                │
│ ──────────────────────────────────  │
│ [OOO] [☆☆☆]                        │ ← グレード
│ ──────────────────────────────────  │
│ CL: 大坪 豊 / SL: 河尻 重和         │
│ 集合: JR三ノ宮駅 20:30              │
│ 申込期限: 3/29(土)                  │
│                         [申込む]    │
└─────────────────────────────────────┘
```

### グレードの見方

**体力度**

| 表示 | 目安 |
|------|------|
| (^^) | ウォーキング程度 |
| O(-) | 軽めのハイキング |
| O | 3〜6時間歩行（六甲高取まで） |
| O(+) | Oより少しハード |
| OO | 6〜8時間歩行（六甲市ヶ原まで） |
| OOO | 8時間以上（六甲全縦） |
| OOOO | 高所・長時間の総合力必要 |

**技術度**

| 表示 | 目安 |
|------|------|
| ☆ | 三点確保で岩場OK |
| ☆☆ | 初級岩登り経験者 |
| ☆☆☆ | 別途条件あり（要確認） |

**岩登り区分**

| 表示 | 対象 |
|------|------|
| 岩A | 初心者向け |
| 岩B | 経験者向け |
| 岩C | CL/SL向け |

### 例会に申し込む

1. 参加したい例会のカードをクリック（詳細を確認）
2. グレード・必要保険口数を確認
3. 「申込む」ボタンをクリック
4. 申込完了（ボタンが「申込取消」に変わる）

> **注意**: 申込期限を過ぎると「締切済」と表示され、申込できません。

### 申込を取り消す

1. 申込済みの例会カードで「申込取消」をクリック
2. 取消完了（ボタンが「申込む」に戻る）

### フィルター機能

- **担当部で絞り込み**: 上部のドロップダウンで部署を選択

## 3.4 📝 計画書

**概要**: 山行計画書を作成・管理します。

### 計画書の作成手順

1. 「+ 新規作成」ボタンをクリック
2. 基本情報を入力
   - **山域**: 例）北アルプス
   - **山名**: 例）槍ヶ岳
   - **目的**: 例）ハイキング
   - **山行形態**: 自主/例会/講座
   - **集合場所・時間**: 例）JR三ノ宮駅 8:00
   - **交通機関**: 例）マイカー
3. 「作成」ボタンをクリック

### ステータスの意味

| ステータス | 意味 |
|-----------|------|
| 下書き | 作成中・未提出 |
| 提出済 | 事務局に提出済み |
| 承認済 | 事務局が確認済み |
| 完了 | 山行終了 |

### 詳細画面で確認できること

- 山域・山名・目的
- 集合場所・交通機関
- 行程（日程ごとの詳細）
- 参加者一覧（役割付き）
- エスケープルート
- 特記事項

---

## 3.5 📖 例会記録

**概要**: 例会の記録を作成・閲覧します。機関誌「すずの子」への掲載用にも使えます。

### 記録の作成手順

1. 「+ 新規作成」ボタンをクリック
2. 必要事項を入力
   - **タイトル**: 例）岩登りA・蓬莱峡 岩登りを楽しもう
   - **日程**: 開催日を選択
   - **天候**: 例）晴れ
   - **アクセス**: 集合〜解散の交通手段
   - **コースタイム**: 行程の時間
   - **本文**: 山行の様子を記述
   - **コース状況**: 道の状態など
   - **特記事項**: ヒヤリハット等
3. 「作成」ボタンをクリック

### ステータスの意味

| ステータス | 意味 |
|-----------|------|
| 下書き | 作成中 |
| 提出済 | 編集担当に提出済み |
| 掲載済 | すずの子に掲載済み |

## 3.6 👥 会員管理

**概要**: 会員情報の登録・検索・編集を行います。

### 会員を検索する

1. 検索欄に名前・メールアドレスを入力
2. 入力に応じて一覧が絞り込まれる

### 会員情報を確認する

1. 会員カードをクリック
2. 詳細モーダルが開く

### 表示される情報

| 項目 | 説明 |
|------|------|
| 氏名・フリガナ | 会員の名前 |
| 性別・生年月日 | 基本情報 |
| 電話・メール | 連絡先 |
| 血液型 | 緊急時の情報 |
| 労山基金 | 加入口数（3口/4口/8口） |
| 入会日 | いつ入会したか |
| ココヘリID | 遭難対策サービスのID |
| 住所 | 居住地 |
| 緊急連絡先 | 家族等の連絡先 |
| 受講済講座 | 完了した講座一覧 |
| 持病 | 健康上の注意事項 |

### 会員を追加する

1. 「+ 会員追加」ボタンをクリック
2. 必要事項を入力
   - 氏名（必須）
   - フリガナ
   - 性別
   - 労山基金口数
   - 電話番号
   - メールアドレス（必須）
3. 「追加」ボタンをクリック

### 労山基金の目安

| 口数 | 参加できる例会 |
|------|---------------|
| 3口 | ハイキング |
| 4口 | 岩A・沢入門 |
| 8口 | アルパイン・雪山・岩BC・沢 |

## 3.7 🎒 装備管理

**概要**: テント・ロープ・無線機などの共同装備を管理します。

### 装備を探す

1. カテゴリで絞り込み（上部ドロップダウン）
   - テント
   - ロープ
   - 無線機
   - 登攀具
   - 調理器具
   - その他

### 装備のステータス

| ステータス | 色 | 意味 |
|-----------|-----|------|
| 貸出可 | 🟢 緑 | 借りられます |
| 貸出中 | 🟡 黄 | 誰かが使用中 |
| メンテナンス中 | 🔴 赤 | 点検・修理中 |

### 装備を借りる

1. 「貸出可」の装備を探す
2. 「貸出」ボタンをクリック
3. 必要事項を入力
   - **貸出先会員**: 自分を選択
   - **返却予定日**: いつまでに返すか
4. 「貸出する」ボタンをクリック

### 装備を返却する

1. 借りている装備を探す
2. 「返却」ボタンをクリック
3. 返却完了（ステータスが「貸出可」に戻る）

### 返却期限超過の場合

- カードが赤い枠で表示される
- 「⚠️ 超過」と表示される
- ダッシュボードにも警告が表示される

# 4. よくある操作シナリオ

## シナリオ1: 例会に参加したい

```
1. 📢 例会案内 を開く
2. 参加したい例会を探す（部署で絞り込み可）
3. カードをクリックして詳細を確認
4. グレード・必要保険口数を確認
5. 「申込む」ボタンをクリック
6. 完了！
```

## シナリオ2: CLとして計画書を作成したい

```
1. 📝 計画書 を開く
2. 「+ 新規作成」をクリック
3. 基本情報を入力（山域・山名・集合場所など）
4. 「作成」をクリック
5. 作成された計画書をクリックして詳細を確認
6. 必要に応じて行程・参加者を追加
7. 完了！
```

## シナリオ3: 山行記録を書きたい

```
1. 📖 例会記録 を開く
2. 「+ 新規作成」をクリック
3. タイトル・日程・天候を入力
4. アクセス・コースタイムを入力
5. 本文に山行の様子を記述
6. コース状況・特記事項を入力
7. 「作成」をクリック
8. 完了！
```

## シナリオ4: 装備を借りたい

```
1. 🎒 装備管理 を開く
2. カテゴリで絞り込み（例: テント）
3. 「貸出可」の装備を探す
4. 「貸出」ボタンをクリック
5. 自分を選択、返却予定日を入力
6. 「貸出する」をクリック
7. 完了！
```

## シナリオ5: 今月の例会予定を確認したい

```
1. 📋 例会企画 を開く
2. カレンダーで今月の予定を確認
3. 部署ごとの色分けで一目瞭然
4. 詳細を知りたい場合は 📢 例会案内 へ
```

# 5. 困ったときは

## よくある質問（FAQ）

### Q: 申込期限を過ぎてしまった

A: システムからの申込はできません。CLに直接連絡してください。

### Q: 自分の保険口数がわからない

A: 👥 会員管理 で自分の名前を検索し、詳細を確認してください。

### Q: 装備の返却期限を過ぎてしまった

A: できるだけ早く返却してください。ダッシュボードに警告が表示されています。

### Q: 過去の例会記録を探したい

A: 📖 例会記録 で一覧を確認できます。タイトルや日程から探してください。

### Q: スマホで見づらい

A: 左上の「☰」メニューからナビゲーションを開けます。
